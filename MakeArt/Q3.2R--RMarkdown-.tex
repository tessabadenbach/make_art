\subsection{Creating a random artwork}\label{creating-a-random-artwork}}

The make\_art() function creates random artworks from three different
templates. To get a random artwork, you just have to call the function:
make\_art()

Then you should see an artwork like this:
\includegraphics{/Users/theresabadenbach/Desktop/myartwork1.png} \#\#
Creating a reproducible artwork If you want to create a reproducible
image, you just need insert an argument when calling the function. For
example, you can call make\_art(100). Then you will get the same image
every time you call make\_art(100).
